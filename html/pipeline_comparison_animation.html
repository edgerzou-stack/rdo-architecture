流水线发送顺序对比动画 (Pipeline Input Sequence)

展示两种不同子块发送顺序对流水线吞吐量的影响。
注：因为 FIFO 物理特性，真实的“气泡”产生于大块接小块 (128 -> 64)的降级瞬间，而非小接大。

小块 (面积64, 耗时8clk)
中块 (面积128, 耗时16clk)
大块 (面积256, 耗时32clk)
等待气泡 (Stall Bubble)
序列 1: 将小块与中块相对聚集 (1 次大接小) 总耗时: 计算中... clk
顺序: 16*4 -> 4*16 -> 16*8 -> 8*16 -> 16*4 -> 4*16 -> 16*8 -> 8*16 -> 16*8 -> 8*16 -> 16*16
序列 2: 频繁穿插 (2 次大接小) 总耗时: 计算中... clk
顺序: 16*4 -> 16*8 -> 16*8 -> 16*8 -> 16*4 -> 4*16 -> 8*16 -> 8*16 -> 8*16 -> 4*16 -> 16*16
