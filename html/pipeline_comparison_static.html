流水线发送顺序对比示意图 (Pipeline Input Sequence)

展示两种不同子块发送顺序对流水线吞吐量的影响。
注：当大块结束紧跟小块时（大接小），由于硬件特性会在输入端产生气泡停顿。

小块 (面积64)
中块 (面积128)
大块 (面积256)
强制等待气泡 (Bubble)
序列 1: 将小块与中块相对聚集 (1 次大接小)
顺序: 16*4 -> 4*16 -> 16*8 -> 8*16 -> 16*4 -> 4*16 -> 16*8 -> 8*16 -> 16*8 -> 8*16 -> 16*16
序列 2: 频繁穿插 (2 次大接小)
顺序: 16*4 -> 16*8 -> 16*8 -> 16*8 -> 16*4 -> 4*16 -> 8*16 -> 8*16 -> 8*16 -> 4*16 -> 16*16
